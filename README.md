## 🧠 Sobre o projeto

O **EMS7x** é um site de e-commerce focado em moda urbana para jovens (16–35 anos), trazendo roupas modernas e tendências globais com preços acessíveis para Moçambique.

Este projeto foi desenvolvido como uma loja online simples, com foco em:
- Dropshipping (envio direto da China)
- Interface moderna e minimalista
- Facilidade de acesso via celular

---

## 🚀 Funcionalidades

- 🛍️ Catálogo de produtos
- 🎯 Design moderno (preto & branco)
- 🔐 Sistema de login (visual)
- 📱 Botão de contato via WhatsApp
- 💳 Pagamentos locais (M-Pesa, e-Mola)
- 🚚 Informação de envio (15–35 dias)

## 📦 Modelo de negócio

Este projeto utiliza **dropshipping**, onde:
1. O cliente faz o pedido
2. O pagamento é confirmado
3. O produto é encomendado de fornecedores internacionais
4. O fornecedor envia diretamente ao cliente
